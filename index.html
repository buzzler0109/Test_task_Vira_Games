Architecture Overview

Автоматизований процес для рекурсивного копіювання вмісту папок Google Drive зі збереженням ієрархії, динамічною обробкою зображень (watermark) та генерацією фінального звіту.

1. Ініціалізація

Зчитування налаштувань. Створення Data Table в базі n8n для безпечного збору статистики без навантаження на API.

2. Рекурсія (Sub-Workflow)

  • Вузол Loop для послідовної обробки (захист від OOM).
  • Папки: Викликає сам себе (рекурсія).
  • Зображення: Edit Image (динамічні формули) → запис у БД.

3. Звітність

Агрегація вивантажених з БД даних. Створення логу .txt у створеній директорії та відправка нотифікації в Telegram.

⚠️ Масштабування для Продакшену: Рекомендується додати вузол Wait (1 сек) у Loop підпроцесу та Retry On Fail для G-Drive вузлів.
